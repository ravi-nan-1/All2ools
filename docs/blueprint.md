# **App Name**: All2ools

## Core Features:

- Homepage Display: Display 30 AI tool cards in a responsive grid layout with tool name, description, and image.
- Tool Pages: Generate 30 individual tool pages with detailed descriptions, language dropdown, input/output sections, FAQs, and SEO content.
- Language Dropdown: Implement a language dropdown on each tool page with 10 global languages (English, Chinese, Spanish, Hindi, Arabic, French, Bengali, Portuguese, Russian, Urdu) to translate content.
- SEO Content Generation: Automatically generate SEO metadata, JSON-LD schema, and FAQ content for each tool page based on the tool description and functionality; use an AI tool to help provide that SEO content.
- AI Background Remover: Allow users to upload a picture and remove its background
- API Endpoint Latency tool: Create a tool to query a given url and display API latency data, across multiple regions.
- Content Optimizer: Build a tool to examine existing text, and produce advice about SERP intent issues. Use an LLM tool to determine the underlying problems.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent technology, trust, and intelligence.
- Background color: A light gray (#F5F5F5) for a clean and modern look.
- Accent color: A bright green (#90EE90) to highlight calls to action and important elements.
- Body and headline font: 'Inter' for a modern, neutral, and readable feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons to represent different tool categories and functionalities.
- Maintain a clean, spacious, and responsive layout to ensure optimal viewing on all devices.
- Use subtle animations to enhance user interaction and provide feedback, such as loading animations or transitions.
